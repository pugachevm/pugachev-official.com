Privacy policy

Here's described an Instagram Commerce Telegram Bot Privacy policy (the "InstContestBot")

Introduction

InstContestBot is an application which uses interaction between Telegram and Instagram accounts provided by users. The main goals of interaction are safe-subscription on Instagram account @pugachevmark, check the state of user's subscription, re-post link to our Telegram Bot and providing points per each action (the "Services").

The current Privacy policy describes how and when InstContestBot uses collected data to increase points for each User to finally complete the contest. Completed Contest defines by calculating overall points from Users and providing Rewards.

InstContestBot receives information only from our API via our Telegram Bot only. For example, you add our bot via Telegram App, authorize on our Contest using provided by InstContestBot API and receiving Contest informationing messages. So that after authorization on our API User will be accessing to Subscribe on Instagram Account by clicking button which will be sent by InstContestBot.
